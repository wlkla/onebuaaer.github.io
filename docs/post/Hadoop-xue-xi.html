Hadoop学习

  1. 了解集群
  2. 熟悉Hadoop常用命令，注意删除命令的谨慎使用（尽量不用）

了解集群

  1. 集群Cluster为一个产品线内具有同类功能的服务器的集合。

熟悉Hadoop常用命令

  1. hadoop fs –ls/-lsr <path>：列出在指定目录下的文件内容
  2. hadoop fs –du <path>：列出指定文件系统空间总量
  3. 'hadoop fs –mv '：将指定文件移动到指定位置
  4. hadoop fs –cp <src> <dst>：拷贝文件到目标位置
  5. hadoop fs –rm/-rmr [-skipTrash] <src>：删除指定文件
  6. hadoop fs –put/-copyFromLocal/-moveFromLocal <localsrc> <dst>：从本地系统拷贝文件到DFS
  7. hadoop fs –get/–copyToLocal [-ignoreCrc] [-crc] <src> <localdst>：从DFS拷贝文件到本地文件系统
  8. hadoop fs –cat <src>：展示文件内容
  9. hadoop fs –mkdir <path>：在指定位置创建目录
  10. hadoop fs –help [cmd]：帮助命令

总结

今天主要是学习了一些基本命令，感觉除了Hadoop的基本命令除了多了一些固定前缀hadoop fs，逻辑上与Linux命令并没有十分大的区别，因此整体来说学下来并没有太大问题。
